# Projet-DevOps
Ce projet consiste à automatiser le processus de sécurisation d'une application Akaunting à travers un pipeline DevSecOps. Il comprend l'intégration de Nessus pour effectuer un scan de sécurité de l'application en container Docker et l'automatisation de l'application des correctifs via Ansible. Le pipeline est déclenché lors de chaque mise à jour du code sur la branche principale de GitHub. Après avoir vérifié le code et lancé les containers nécessaires (MySQL et Akaunting), un scan de vulnérabilité est effectué avec Nessus, et les résultats sont stockés. Ensuite, un playbook Ansible est utilisé pour appliquer automatiquement les mises à jour et correctifs nécessaires sur les containers, assurant ainsi une gestion continue de la sécurité et de la mise à jour de l'application dans un environnement DevSecOps.
